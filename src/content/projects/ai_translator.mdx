Dette prosjektet kom fra et behov for å oversette flere sider i et emne i læringsplattformen fra norsk til engelsk. Løsningen ble å bruke en KI modell fra Hugging Face for å oversette teksten, og deretter bruke Canvas LMS API for å oppdatere sidene. Det ble laget et script i Tampermonkey som henter innholdet fra sidene, og så sender det til en Express server hvor selve oversettingen foregår. Resultatet sendes så tilbake til brukerscriptet som bruker læringsplattformens API for å oppdatere sidene igjen.
